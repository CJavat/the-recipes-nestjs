## Dev

1. Instalar las dependencias:
```
npm install
```
2. Renombrar el archivo `.env.template` por `.env`.
3. Configuar las variables de entorno.
4. Correr la base de datos:
```
docker compose up -d
```
5. Arrancar el proyecto en modo desarrollo: 
```
npm run dev
```
